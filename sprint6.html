for GoodLife Fitness

Objective

Dataset Overview

Column Names:
- review_id: Unique identifier for each review.
- review_text: Content of the review.
- upvotes: Number of upvotes received.
- app_version: Version of the app being reviewed.
- posted_date: Date and time when the review was posted.

Post-Normalization Column Names:

- review_id
- review_text
- upvotes
- app_version
- posted_date

Analysis Process

Sentiment Analysis Function

The sentiment analysis was performed using `TextBlob`. The function `analyze_sentiment` classifies the sentiment of each review text as follows:
- Positive: Sentiment polarity score > 0
- Negative: Sentiment polarity score < 0
- Neutral: Sentiment polarity score = 0
